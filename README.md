# SimpleShell
## OS Assignment 2
- Simple Shell made and implemented in C, it waits for user input and executes the command along with arguments usnig execvp.
- Stops on ctrl+c
- fork() system call is also used.
- SIGINT handler function is made (to terminate on ctrl+c and show few extra details)
- SIGCHLD handler function is made for background processes (bonus part)
- Pipes are also implemented.
